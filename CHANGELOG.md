# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-30

### Added
- Initial release
- Single image conversion support
- Multi-format support (HEIC, JPEG, PNG, WebP, GIF, BMP, TIFF)
- Drag-and-drop file upload
- Image preview functionality
- Batch conversion UI (backend support ongoing)
- Express.js backend with multer file handling
- Sharp and heic-convert integration for image processing
- Responsive web interface with Font Awesome icons

### Features
- Convert images to JPEG, PNG, WebP, GIF, HEIC, HEIF, or TIFF
- Real-time image preview before conversion
- Easy file upload with visual feedback
- Privacy-first: all processing done on your server
