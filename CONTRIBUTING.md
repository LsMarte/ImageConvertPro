# Contributing to Image Converter Pro

Thank you for your interest in contributing! We welcome contributions from everyone.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/heic-converter.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit with clear messages: `git commit -m 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature-name`
7. Submit a Pull Request

## Code Guidelines

- Follow existing code style and conventions
- Write meaningful commit messages
- Test your changes locally before submitting
- Update documentation if needed
- Keep PRs focused and reasonably sized

## Development Setup

```bash
# Install dependencies
cd heic-converter
npm install

# Run in development mode
npm run dev

# Run tests
npm test
```

## Reporting Bugs

- Use the issue tracker to report bugs
- Include OS, Node.js version, and browser information
- Provide steps to reproduce the issue
- Include error messages and screenshots if applicable

## Feature Requests

- Check existing issues to avoid duplicates
- Clearly describe the feature and its use case
- Provide examples if possible

## Questions?

Feel free to open an issue with the `question` label or start a discussion.

Thank you for contributing!
